# Scindapsus

- [lock]：封装spring官方实现的分布式锁，支持注解及spel

- [log]：一行代码引入日志脱敏、请求日志打印

- [surl]：一键配置短网址服务

- [drools]：四个类封装drools,简化为工具类调用

- [apt]：使用Java Annotation Processing Tool快速生成JPARepository

- [graalvm]：使用graalvm调用其他语言方法
